Informazioni generali

Esplora facilmente le pagine web realizzate durante il triennio di Informatica all'ITIS S.Cannizzaro di Rho. All'interno di questo sito Web è possibile visitare facilmente le pagine realizzate e scoprire le molteplici sfaccettature del linguaggio HTML. Il linguaggio HTML è infatti il metodo più gettonato per progettare le pagine Web e per gestirne lo stile. Cliccando su "Esplora" nella Homepage del sito, riuscirai a visualizzare le pagine web ordinate in relazione all'anno di creazione. il design di navigazione è stato reso semplice ed intuitivo per permettere a qualsiasi utente di navigare senza riscontrare alcun problema.
